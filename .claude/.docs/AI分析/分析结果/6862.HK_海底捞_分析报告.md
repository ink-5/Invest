## 五、支柱一：存量资产垫评估

### 5.1 T0 计算

```
现金池 = 现金等价物(47.56亿) + 短期理财(53亿) = 100.56亿元
T0 = 现金池 − 总负债(116亿) = 100.56 − 116 = -15.44亿元
T0_NAV = -15.44亿 / 55.74亿股 = -0.28元/股 ≈ -0.30港元/股
当前股价 = 19.38港元
判定：T0_NAV < 0 → T0不达标 ❌
```

### 5.2 T1 计算

```
有息负债 = 长期借款等(21亿元)
T1 = 现金池 − 有息负债 = 100.56 − 21 = 79.56亿元
T1_NAV = 79.56亿 / 55.74亿股 = 1.43元/股 ≈ 1.53港元/股
判定：T1_NAV(1.53) << 股价(19.38) → T1不达标 ❌
```

### 5.3 T2 计算

```
T2 = 现金等价物×1.0(47.56亿) + 短期理财×1.0(53亿)
    + 应收账款×0.85(1.73×0.85=1.47亿)
    + 存货×0.8(13.39×0.8=10.71亿)
    + 其他流动资产×0.5(估算50×0.5=25亿)
    − 总负债(116亿)
    = 137.74 − 116 = 21.74亿元
T2_NAV = 21.74亿 / 55.74亿股 = 0.39元/股 ≈ 0.42港元/股
判定：T2_NAV(0.42) << 股价(19.38) → T2不达标 ❌
```

存货折扣系数选择0.8（餐饮行业食材存货，周转较快，报废率相对较低）

### 5.4 特殊科目处理

- 受限现金：未发现重大受限现金
- 合同负债：餐饮行业预收款较少，影响有限
- 租赁负债：海底捞作为连锁餐饮企业，使用权资产约29.54亿元，对应租赁负债规模较大（IFRS 16），已包含在总负债中
- 注意：若将租赁负债单独计入有息负债，T1_NAV将进一步下降
