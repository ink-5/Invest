## 五、支柱一：存量资产垫评估

### 5.1 T0 计算

```
现金池 = 银行结存及定期存款(36,069) − 受限制银行结存(509) = 35,560 百万港元
T0 = 现金池 − 总负债(101,581) = 35,560 − 101,581 = -66,021 百万港元
T0_NAV = -66,021 / 3,500 = -HK$18.86/股
当前股价 = HK$46.80
判定：T0_NAV(-18.86) < 0 → ❌ T0不达标（现金远不足以覆盖总负债）
```

使用H1 2025数据验证：
```
现金池 = 33,005 − 576 = 32,429
T0 = 32,429 − 108,762 = -76,333
T0_NAV = -76,333 / 3,500 = -HK$21.81/股 → ❌ 更差
```

### 5.2 T1 计算

```
有息负债 = 短期借款(11,139) + 长期借款(41,577) = 52,716 百万港元
  注：另有租赁负债4,916百万，保守起见纳入
  广义有息负债 = 52,716 + 4,916 = 57,632
T1 = 现金池(35,560) − 有息负债(52,716) = -17,156 百万港元
T1_NAV = -17,156 / 3,500 = -HK$4.90/股
  若含租赁负债：T1 = 35,560 − 57,632 = -22,072 → T1_NAV = -HK$6.31/股
判定：T1_NAV(-4.90) < 0 → ❌ T1不达标
```

### 5.3 T2 计算

```
长实集团为地产综合企业，流动资产构成特殊：
- 流动资产合计 = 175,917 百万港元（FY2024）
- 其中：银行结存36,069 + 应收等7,901 + 应收贷款2,171 + 待售物业129,776

T2 = 现金等价物×1.0(35,560) + 应收账款×0.85(7,901×0.85=6,716)
   + 存货(待售物业)×0.7(129,776×0.7=90,843) + 应收贷款×0.5(2,171×0.5=1,086)
   − 总负债(101,581)
   = 35,560 + 6,716 + 90,843 + 1,086 − 101,581 = +32,624 百万港元
T2_NAV = 32,624 / 3,500 = +HK$9.32/股
判定：T2_NAV(+9.32) < 股价(46.80) → ❌ T2不达标（T2_NAV远低于股价）

若存货按更保守的0.5折扣：
  T2_保守 = 35,560 + 6,716 + 64,888 + 1,086 − 101,581 = +6,669
  T2_NAV_保守 = +HK$1.91/股 → 仍远低于股价
```

存货折扣系数选取理由：地产发展中物业在当前香港楼市下行期，采用0.7折扣系数（长实项目以香港/海外优质地段为主，接近完工项目占比较高）。保守情景下采用0.5。

### 5.4 特殊科目处理

- **受限现金**：HK$5.09亿（FY2024），占银行结存总额36,069的1.4%。根据内地房地产发展要求，物业销售所得存放于银行并受限使用。受限比例≤5%，可忽略，正常计入现金池。✅
- **合同负债**：客户订金HK$107.66亿（FY2024），为物业预售款。地产公司合同负债需交付物业才能确认收入，不等同于白酒行业的准现金，T0/T1中不加入现金池。
- **租赁负债**：HK$49.16亿（FY2024），IFRS 16下经营租赁已上表（主要为Greene King酒吧物业租赁），保守起见纳入广义有息负债。
- **投资物业**：HK$1,507.08亿（FY2024），按公允价值计量，有变现价值但不属于流动资产，不纳入T0/T1/T2计算。
- **已抵押物业**：HK$311.74亿（FY2024），其中物业发展及投资抵押HK$64.14亿，Greene King业务其他借款抵押HK$247.60亿。核心资产有一定抵押但属行业常规。
- **永久资本证券**：HK$79.29亿，无固定到期日，列为权益而非负债。

## 十、操作建议

### 10.1 烟蒂股框架下的结论

**当前不建议按烟蒂股策略建仓。** 原因：

1. **支柱一不达标**：T0/T1资产垫为负值，T2虽为正（+HK$9.32/股）但远低于股价HK$46.80
2. **支柱三不达标**：股息率3.68%低于港股6%门槛，不满足A型核心条件；净负债状态不满足B型条件
3. **估值已修复**：股价过去一年从HK$28涨至HK$47（+67%），大部分深度折价已被市场修正
4. **股价已超分析师目标价**：当前HK$46.80已超过TipRanks平均目标价HK$40.66

### 10.2 观察池条件（重新触发分析的价格）

| 触发条件 | 对应股价 | 说明 |
|:---------|:--------:|:-----|
| 股息率回到6% | ≤ HK$29.0 | 假设维持HK$1.74/股派息 |
| PB回到0.30 | ≤ HK$33.8 | 假设每股净资产HK$112.65不变 |
| 股息率回到6% + PB≤0.30 | ≤ HK$29.0 | 同时满足A型两个核心条件 |

> 即使股价回落至上述水平，支柱一（资产垫）仍是结构性障碍。长实作为重资产地产公司，天然不适合烟蒂股框架。

### 10.3 替代分析框架建议

长实集团更适合以下分析框架：

1. **NAV重估法**：对物业资产逐项重估（投资物业按市场租金资本化率、发展中物业按预期售价折现），计算重估NAV与市值的折价
2. **股息折现模型（DDM）**：基于稳定的派息历史和高FCF覆盖率，用DDM估算内在价值
3. **SOTP分部估值**：分别估值物业发展、物业投资、Greene King、长江基建持股，加总后与市值比较
4. **传统价值投资**：关注PB修复+股息收入的总回报，而非烟蒂股框架的"现金资产覆盖市值"逻辑

### 10.4 收益周期计算（参考性，非烟蒂股框架）

#### 预期收益率精确计算

| 情景 | PB目标 | 资本利得 | 持有期 | 累计税后股息 | **总回报** | **年化IRR** |
|:-----|:------:|:--------:|:------:|:----------:|:----------:|:-----------:|

```

  目标PB = 0.50 → 目标股价 = 112.65 × 0.50 = HK$56.33
```

## 十一、风险提示

### 11.1 该标的特定风险

1. **香港楼市持续下行风险**：香港住宅价格自2021年高点已累计下跌约25-30%，若进一步下跌，长实的物业发展利润和投资物业公允价值将继续承压，每股净资产可能下降
2. **内地物业敞口风险**：长实在内地多个城市有物业发展项目，内地楼市低迷可能导致项目去化缓慢、利润率下降
3. **Greene King经营风险**：英国酒吧业务面临消费疲软、人工成本上升、能源成本波动等压力。FY2023曾发生商誉减值HK$19.63亿
4. **利率环境风险**：虽然有息负债率低（10.5%），但HK$527亿的绝对债务规模仍使公司对利率变动敏感
