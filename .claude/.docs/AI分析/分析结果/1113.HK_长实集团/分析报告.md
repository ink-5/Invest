## 五、支柱一：存量资产垫评估

### 5.1 T0 计算

```
现金池 = 现金等价物(34,182) + 短期理财(1,378) = 35,560 百万港元
T0 = 现金池 − 总负债(101,581) = 35,560 − 101,581 = -66,021 百万港元
T0_NAV = -66,021 / 3,500 = -HK$18.86/股
当前股价 = HK$46.80
判定：T0_NAV(-18.86) < 0 → ❌ T0不达标（现金远不足以覆盖总负债）
```

### 5.2 T1 计算

```
有息负债 = 短期借款(11,139) + 长期借款(41,577) = 52,716 百万港元
  注：Total Debt报告为57,632百万，差额4,916百万为租赁负债等，保守起见使用Total Debt
T1 = 现金池(35,560) − 有息负债(57,632) = -22,072 百万港元
T1_NAV = -22,072 / 3,500 = -HK$6.31/股
判定：T1_NAV(-6.31) < 0 → ❌ T1不达标（现金不足以覆盖有息负债）
```

### 5.3 T2 计算

```
长实集团为地产综合企业，流动资产构成特殊：
- 流动资产合计 = 175,917 百万港元
- 其中：现金35,560 + 应收账款2,114 + 存货/发展中物业129,776 + 其他流动资产8,467

T2 = 现金等价物×1.0(35,560) + 应收账款×0.85(2,114×0.85=1,797)
   + 存货(发展中物业)×0.7(129,776×0.7=90,843) + 其他流动资产×0.5(8,467×0.5=4,234)
   − 总负债(101,581)
   = 35,560 + 1,797 + 90,843 + 4,234 − 101,581 = +30,853 百万港元
T2_NAV = 30,853 / 3,500 = +HK$8.81/股
判定：T2_NAV(+8.81) < 股价(46.80) → ❌ T2不达标（T2_NAV远低于股价）

注：虽然T2_NAV为正值，但仅HK$8.81/股，远低于当前股价HK$46.80，
不满足"T2_NAV > 股价"的达标条件。
若存货按更保守的0.5折扣：
  T2_保守 = 35,560 + 1,797 + 64,888 + 4,234 − 101,581 = +4,898
  T2_NAV_保守 = +HK$1.40/股 → 仍远低于股价
```

存货折扣系数选取理由：地产发展中物业在当前香港楼市下行期，采用0.7折扣系数（长实项目以香港/海外优质地段为主，接近完工项目占比较高）。保守情景下采用0.5。

### 5.4 特殊科目处理

- **受限现金**：⚠️ 数据缺失，无法评估。地产公司通常有部分预售监管资金受限。
- **合同负债**：地产公司合同负债（预售楼款）金额较大，但在T0/T1中不加入现金池（因为需要交付物业才能确认收入，不等同于白酒行业的准现金）。
- **租赁负债**：IFRS 16下经营租赁已上表，应纳入有息负债。长实的租赁负债相对较小（酒店/酒吧业务为主），已包含在总负债中。
- **投资物业**：长实持有大量投资物业（按公允价值计量），这些资产有变现价值但不属于流动资产，不纳入T0/T1/T2计算。

### 5.5 资产垫分级结论

| 等级 | NAV/每股 | vs 股价(HK$46.80) | 判定 |
|:----:|:--------:|:-------:|:----:|
| T0 | -HK$18.86 | 远低于0 | ❌ |
| T1 | -HK$6.31 | 远低于0 | ❌ |
| T2 | +HK$8.81 | 远低于股价 | ❌ |

**最终资产垫等级：不达标**

> 💡 **核心原因**：长实集团是资产密集型地产企业，其价值主要体现在投资物业（按公允价值数百亿港元）、发展中物业（HK$1,298亿）、土地储备等非流动/半流动资产上，而非现金类资产。T2虽为正值（+HK$8.81/股），但远低于股价HK$46.80，不满足"T2_NAV > 股价"的达标条件。烟蒂股策略要求"现金类资产覆盖市值"，这一框架天然不适用于重资产地产公司。长实的低PB（0.41）反映的是市场对地产资产的折价，而非现金资产被低估。

## 十、操作建议

### 10.1 烟蒂股框架下的结论

**当前不建议按烟蒂股策略建仓。** 原因：

1. **支柱一不达标**：T0/T1资产垫为负值，T2虽为正（+HK$8.81/股）但远低于股价HK$46.80
