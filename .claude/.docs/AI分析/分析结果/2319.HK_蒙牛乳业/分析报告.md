## 二、商业模式深度扫描

### 2.1 赚钱逻辑（Core Business Model）

| 维度 | 分析内容 |
|:-----|:---------|
| 核心主业 | 乳制品生产及经销，涵盖液态奶（常温奶、低温奶）、冰淇淋、奶粉（含贝拉米婴幼儿奶粉）、奶酪等品类 |
| 收入构成 | 液态奶为绝对现金牛（占收入约80%+），冰淇淋为第二大品类，奶粉业务（贝拉米）持续亏损并大幅减值 |
| 盈利模式 | 赚取生产加工差价——从上游牧场采购原奶，加工为成品乳制品后通过经销商/零售渠道销售，赚取品牌溢价和规模效应带来的毛利 |
| 价值链位置 | 中游加工环节，上游通过控股现代牧业(1117.HK)实现部分原奶自给，下游依赖经销商网络和零售终端 |
| 客户结构 | To C为主（终端消费者），通过经销商体系分销，客户分散度高 |

### 2.2 周期性判别（Cyclicality Assessment）

| 维度 | 判断 |
|:-----|:-----|
| 行业周期属性 | 弱周期（乳制品属日常必需消费品，需求刚性较强） |
| 判定依据 | 乳制品消费与GDP相关性较低，需求弹性小；但行业存在供需周期（原奶价格波动） |
| 当前周期阶段 | 衰退/调整期——2024年行业供需矛盾突出，原奶过剩导致上游牧场亏损，消费信心不足导致终端需求疲软，收入同比下降10.1% |
| 对烟蒂股分析的影响 | 当前处于行业低谷期，存货和应收减值风险相对可控（乳制品保质期短，存货周转快），但盈利能力受压 |

### 5.3 T0 计算（最严格：现金池 − 总负债）

```
T0 = 含合约负债的现金池 − 总负债
   = 42,429,100 − 58,267,301
   = −15,838,201千元（约−158.38亿RMB）

T0_NAV = −15,838,201 / 3,910,000千股 = −4.05 RMB/股

当前股价 = 15.720 HKD × 0.91 = 14.31 RMB

判定：T0_NAV（−4.05 RMB）< 0 < 股价（14.31 RMB）→ ❌ T0严重不达标
```

**结论：** ❌ **T0完全不达标。现金池扣除全部负债后为负值，不存在任何T0级资产垫保护。**

### 5.4 T1 计算（中等：现金池 − 有息负债）

```
T1 = 含合约负债的现金池 − 有息负债
   = 42,429,100 − 34,637,200
   = 7,791,900千元（约77.92亿RMB）

T1_NAV = 7,791,900 / 3,910,000千股 = 1.99 RMB/股

当前股价 = 15.720 HKD × 0.91 = 14.31 RMB

判定：T1_NAV（1.99 RMB）<< 股价（14.31 RMB）→ ❌ T1严重不达标
      T1净现金77.92亿 vs 市值554亿 → 净现金仅为市值的14.0%
```

**结论：** ❌ **T1不达标。虽然扣除有息负债后有正净现金77.92亿，但远低于市值588亿。**

### 5.5 T2 计算（宽松：打折流动资产 − 总负债）

```
存货折扣系数选择：0.75（乳制品保质期短，但属消费品，报废率中等）

T2_可变现流动资产：
  = 现金池×1.0 + 应收账款×0.85 + 存货×0.75 + 预付款等×0.50
  = 39,587,479×1.0 + 3,261,858×0.85 + 4,936,666×0.75 + 1,979,279×0.50
  = 39,587,479 + 2,772,579 + 3,702,500 + 989,640
  = 47,052,198千元

T2 = T2_可变现流动资产 − 总负债
   = 47,052,198 − 58,267,301
   = −11,215,103千元（约−112.15亿RMB）

T2_NAV = −11,215,103 / 3,910,000千股 = −2.87 RMB/股

判定：T2_NAV（−2.87 RMB）< 0 < 股价（14.31 RMB）→ ❌ T2严重不达标
```

**结论：** ❌ **T2完全不达标。即使将流动资产打折变现，仍无法覆盖总负债。**
