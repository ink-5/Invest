## 五、支柱一：存量资产垫评估

> 以下计算基于2025年6月30日中期报告数据（最新），单位：百万人民币。汇率按1港元≈0.915人民币换算。

### 5.1 T0 计算

```
现金池 = 银行结余及现金(5,871) + 其他金融资产-流动(1,977) + FVTPL金融资产-流动(1,225) + 受限银行存款-流动(144) = 9,217百万元
  注：受限银行存款占现金池比例 = 144/9,217 = 1.6% ≤ 5%，可忽略，正常计入
总负债 = 11,551百万元
T0 = 现金池 − 总负债 = 9,217 − 11,551 = -2,334百万元
T0_NAV = -2,334 / 5,415百万股 = -0.43元/股 ≈ -0.47港元/股
当前股价 = 16.87港元
判定：T0_NAV < 0 → T0不达标 ❌
```

### 5.2 T1 计算

```
有息负债 = 银行借款(79) + 长期债券-流动(2,066) + 长期债券-非流动(0) + 租赁负债-流动(888) + 租赁负债-非流动(2,843)
         = 5,876百万元
  其中：纯银行借款+债券 = 79 + 2,066 = 2,145百万元
  租赁负债 = 888 + 2,843 = 3,731百万元

保守口径（含租赁负债，IFRS 16要求）：
T1 = 现金池(9,217) − 有息负债(5,876) = 3,341百万元
T1_NAV = 3,341 / 5,415 = 0.62元/股 ≈ 0.68港元/股
判定：T1_NAV(0.68港元) << 股价(16.87港元) → T1不达标 ❌

宽松口径（仅银行借款+债券，不含租赁负债）：
T1_宽松 = 现金池(9,217) − 银行借款+债券(2,145) = 7,072百万元
T1_NAV_宽松 = 7,072 / 5,415 = 1.31元/股 ≈ 1.43港元/股
判定：T1_NAV(1.43港元) << 股价(16.87港元) → 仍然不达标 ❌
```

### 5.3 T2 计算

```
存货折扣系数：0.8（餐饮行业食材存货，周转较快）

T2 = 现金池×1.0(9,217)
    + 贸易及其他应收款项×0.85(1,163 × 0.85 = 989)
    + 应收关联方×0.85(309 × 0.85 = 263)
    + 存货×0.8(902 × 0.8 = 722)
    + 其他流动资产×0.5(租赁按金96 + 持作出售资产111 = 207 × 0.5 = 104)
    − 总负债(11,551)
T2 = 9,217 + 989 + 263 + 722 + 104 − 11,551 = -256百万元
T2_NAV = -256 / 5,415 = -0.05元/股 ≈ -0.05港元/股
判定：T2_NAV < 0 → T2不达标 ❌
```

### 5.4 特殊科目处理

- **租赁负债**：海底捞作为连锁餐饮企业，IFRS 16下使用权资产29.42亿元、租赁负债37.31亿元。这是餐饮连锁企业的结构性特征，保守口径应纳入有息负债
- **受限现金**：2025.6.30受限银行存款1.46亿元，占现金池1.6%，≤5%可忽略 ✅
- **合约负债**：9.37亿元，主要为会员积分计划(8.00亿)和预付卡(1.29亿)。餐饮行业合约负债不同于白酒行业的预收款，不加入现金池
- **长期债券**：USD 2.855亿美元债券（2026年1月到期），已全部重分类为流动负债，计入有息负债
- **持作出售资产**：1.11亿元（对应负债0.56亿元），为拟处置的餐厅业务
