## 五、支柱一：存量资产垫评估

### 5.1 T0 计算

```
现金池 = 现金及等价物(58.71亿) + 短期理财(19.77亿) + 其他现金类(12.25亿) = 90.73亿元
总负债 = 115.51亿元
T0 = 现金池 − 总负债 = 90.73 − 115.51 = -24.78亿元
T0_NAV = -24.78亿 / 55.7亿股 = -0.44元/股 ≈ -0.48港元/股
当前股价 = 16.87港元
判定：T0_NAV < 0 → T0不达标 ❌
```

### 5.2 T1 计算

```
有息负债 = Total Debt 58.76亿元（含租赁负债）
  注：若仅计银行借款（流动部分21.45亿 + 长期0），有息负债 = 21.45亿元
  保守口径（含租赁负债）：
T1 = 现金池(90.73) − 有息负债(58.76) = 31.97亿元
T1_NAV = 31.97亿 / 55.7亿股 = 0.57元/股 ≈ 0.63港元/股
判定：T1_NAV(0.63) << 股价(16.87) → T1不达标 ❌

  宽松口径（仅银行借款）：
T1_宽松 = 现金池(90.73) − 银行借款(21.45) = 69.28亿元
T1_NAV_宽松 = 69.28 / 55.7 = 1.24元/股 ≈ 1.36港元/股
判定：T1_NAV(1.36) << 股价(16.87) → 仍然不达标 ❌
```

### 5.3 T2 计算

```
T2 = 现金等价物×1.0(90.73亿)
    + 应收账款×0.85(估算少量，忽略)
    + 存货×0.8(估算含在流动资产中)
    + 其他流动资产×0.5(估算)
    − 总负债(115.51亿)
近似计算：流动资产合计117.98亿 × 0.7 − 总负债115.51亿
    = 82.59 − 115.51 = -32.92亿元
T2_NAV = -32.92亿 / 55.7亿股 = -0.59元/股 ≈ -0.65港元/股

更精确计算（分项折扣）：
  现金池×1.0 = 90.73亿
  非现金流动资产(117.98-90.73=27.25亿) × 0.65 = 17.71亿
  合计可变现 = 108.44亿
  T2 = 108.44 − 115.51 = -7.07亿元
  T2_NAV = -7.07 / 55.7 = -0.13元/股 ≈ -0.14港元/股
判定：T2_NAV < 0 → T2不达标 ❌
```

存货折扣系数选择0.8（餐饮行业食材存货，周转较快）

### 5.4 特殊科目处理

- **租赁负债**：海底捞作为连锁餐饮企业，IFRS 16下使用权资产和租赁负债规模较大。Total Debt 58.76亿中包含大量租赁负债，这是餐饮连锁企业的结构性特征
- **受限现金**：未发现重大受限现金
- **合同负债**：餐饮行业预收款较少，影响有限
