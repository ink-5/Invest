## 五、支柱一：存量资产垫评估

### 5.1 T0 计算

```
现金池（2025中期）= 现金及等价物 + 可变现投资 = 1,372.68亿港元
总负债（2024年报）= 4,599.5亿港元

T0 = 现金池 − 总负债 = 1,372.68 − 4,599.5 = −3,226.82亿港元
T0_NAV = −3,226.82 / 38.30 = −84.26港元/股

当前股价 = 63.20港元

判定：T0_NAV (−84.26) < 股价 (63.20) → ❌ T0不达标
```

### 5.2 T1 计算

```
有息负债（2025中期）= 2,565.89亿港元

T1 = 现金池 − 有息负债 = 1,372.68 − 2,565.89 = −1,193.21亿港元
T1_NAV = −1,193.21 / 38.30 = −31.16港元/股

判定：T1_NAV (−31.16) < 股价 (63.20) → ❌ T1不达标
```

### 5.3 T2 计算

```
T2计算（采用2024年报数据，单位：亿港元）：

流动资产（估算）：
  现金等价物×1.0 = 1,213.0
  应收账款×0.85 = 141.32 × 0.85 = 120.12
  存货×0.70 = 238.47 × 0.70 = 166.93（零售业存货折扣70%）
  其他流动资产×0.5 ≈ 200（保守估计）
  ────────────────────────────
  T2_可变现流动资产 ≈ 1,700.05亿港元

T2_净资产 = 1,700.05 − 4,599.5 = −2,899.45亿港元

⚠️ 上述计算基于不完整数据，采用替代方案：

替代方案 - 基于账面净资产（含商誉调整）：
  账面净资产 = 6,525.9亿港元
  商誉及无形资产调整 = −(2,687 + 1,964) × 50% = −2,325.5亿（保守50%折扣）
  调整后净资产 ≈ 4,200.4亿港元

  调整后NAV = 4,200.4 / 38.30 = 109.66港元/股

判定：调整后NAV (109.66) > 股价 (63.20) → ✅ T2达标（保守估计）
```

**说明**：长和作为大型综合企业，商誉和无形资产规模较大（约4,651亿港元），占总资产约42%。这主要是多年并购积累所致。保守起见，对商誉+无形资产打50%折扣后，调整后NAV仍显著高于当前股价。
