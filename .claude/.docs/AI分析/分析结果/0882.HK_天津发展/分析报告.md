## 五、支柱一：存量资产垫评估

### 5.1 T0 计算（最严格）

```
现金池 = 2,444,547 + 1,788,300 + 1,617,398 + 775,603 + 1,064,449 + 602,611 + 59,350
       = 8,352,258千港元（83.52亿港元）

T0 = 现金池 − 总负债 = 8,352,258 − 4,761,783 = 3,590,475千港元（35.90亿港元）
T0_NAV = 35.90亿 / 10.725亿股 = 3.349港元/股

当前股价 = 2.71港元
判定：T0_NAV(3.349) > 股价(2.71) → ✅ T0达标
安全边际 = (35.90 − 29.06) / 35.90 = 19.1%
```

> 注：相比上次分析（股价1.960，安全边际41.4%），安全边际收窄至19.1%，但仍满足T0级。

### 5.2 T1 计算

```
有息负债 = 银行贷款(1,942,187 + 30,611) + 租赁负债(19,645) = 1,992,443千港元（19.92亿港元）
T1 = 现金池 − 有息负债 = 8,352,258 − 1,992,443 = 6,359,815千港元（63.60亿港元）
T1_NAV = 63.60亿 / 10.725亿股 = 5.930港元/股

判定：T1_NAV(5.930) > 股价(2.71) → ✅ T1达标
T1净现金是市值的2.19倍
```

### 5.3 T2 计算

存货折扣系数：公用事业/医药存货（低报废率）取0.80

```
T2 = 现金池×1.0 + 贸易应收×0.85 + 其他应收×0.70 + 存货×0.80
     + 合约资产×0.85 + 关联方应收×0.90 + 融资租赁应收×0.85 − 总负债
```

### 6.3 经营现金流为负的原因分析

FY2024经营现金流为负（−7.23亿港元）的主因：

1. **联营企业利润非现金确认**：权益法确认联营利润4.19亿港元，但未收到等额现金（需等待联营企业分红）
2. **营运资本变动**：应收款增加、结构性存款/信托存款配置增加
3. **银行贷款结构调整**：FY2024将17.1亿非流动贷款重分类为流动贷款（到期偿还），影响现金流分类

> ⚠️ **重要判断**：经营现金流为负并非因为核心业务亏损（FY2024净利润8.01亿港元），而是控股公司结构性特征（联营利润非现金化）和现金管理活动所致。2025H1经营现金流已转正，趋势向好。

**支柱二综合结论：⚠️ 有条件通过 — FY2024严格不达标，但2025H1显著改善，核心业务盈利能力无忧，T0级资产垫提供充足缓冲。**

## 十一、风险提示

### 11.1 该标的特定风险

1. **支柱二现金流风险**：FY2023和FY2024连续两年经营活动现金流为负，虽2025H1转正，但全年趋势尚未确认。若FY2025全年CFO仍为负，资产垫将持续被消耗。

5. **FVTOCI公允价值波动**：天士力集团FVTOCI从FY2024末12.73亿降至2025H1的8.52亿（−33.0%），非上市投资公允价值存在不确定性。
