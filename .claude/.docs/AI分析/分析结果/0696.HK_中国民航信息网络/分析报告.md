## 二、商业模式深度扫描

### 2.1 赚钱逻辑（Core Business Model）

| 维度 | 分析内容 |
|:-----|:---------|
| 核心主业 | 中国民航业唯一的GDS（全球分销系统）和核心IT基础设施提供商，为航空公司、机场、旅行社提供信息技术服务 |
| 收入构成 | 航空信息技术服务（核心，含订座系统、离港系统等）约占45-50%；结算清算服务约占15-20%；机场信息技术服务约占25-27%；分销信息技术服务及其他约占10-15% |
| 盈利模式 | 按交易量收费（每笔订座/离港/结算收取固定费用）+ 系统集成项目收入 + 数据网络服务费。本质是"民航交易税"模式 |
| 价值链位置 | 处于民航产业链的核心中枢节点，连接航空公司、机场、旅行社、旅客的信息枢纽 |
| 客户结构 | To B为主。服务中国全部38家商业航空公司 + 350+家境外航空公司 + 8000+家旅行社。客户高度集中于国有航空公司（国航、南航、东航等同时也是股东） |

### 2.2 周期性判别（Cyclicality Assessment）

| 维度 | 判断 |
|:-----|:-----|
| 行业周期属性 | 弱周期。航空IT基础设施需求与航空客运量正相关，但具有刚性特征——航空公司无论盈亏都必须使用订座和离港系统 |
| 判定依据 | COVID期间（2020-2022）收入从81亿降至52亿（-36%），但未亏损；2024年已恢复至88亿（超疫情前水平）。波动性低于航空公司本身 |
| 当前周期阶段 | 复苏后期/稳定期。中国民航客运量已超过2019年水平，公司收入和利润均创历史新高 |
| 对烟蒂股分析的影响 | 当前处于盈利高点附近，资产负债表健康，但估值也已充分反映复苏——PB 1.2x不存在低估 |

### 2.3 提价权分析（Pricing Power）

| 维度 | 评估 |
|:-----|:-----|
| 定价能力 | 价格制定者（Price Maker）。作为垄断供应商，定价受民航局指导但具有较强议价地位 |
| 判定依据 | 毛利率长期维持在45-50%水平，营业利润率FY2024达27.2%、H1 2025达43.4%，显示极强的定价能力 |
| 成本转嫁能力 | 强。主要成本为人工和折旧，非原材料驱动，成本结构稳定 |
| 通胀环境下的表现 | 良好。IT服务费随交易量增长自然提升，人工成本上升可通过效率提升消化 |

### 2.4 护城河识别（Moat Analysis）

| 护城河类型 | 是否具备 | 强度评估 | 说明 |
|:-----------|:--------:|:--------:|:-----|
| 品牌效应 | 否 | 无 | B2B业务，品牌不构成壁垒 |
| 网络效应 | 是 | 强 | 连接航空公司-机场-旅行社的双边网络，参与者越多价值越大 |
| 转换成本 | 是 | 强 | 航空公司更换核心IT系统的成本极高，涉及安全认证、数据迁移、人员培训 |
| 规模经济 | 是 | 强 | 边际成本极低，交易量增长直接转化为利润 |
| 牌照/准入壁垒 | 是 | 强 | 民航局指定的唯一GDS供应商，政策性垄断地位 |
| 成本优势 | 是 | 中 | 规模效应带来的成本优势 |
| **综合护城河评级** | | **宽（Wide Moat）** | 政策垄断+网络效应+高转换成本，护城河极宽 |

> 💡 **对烟蒂股的特殊意义**：该公司拥有极宽护城河，资产价值高度可防御。但正因如此，市场给予了合理甚至偏高的估值（PB>1），使其不符合烟蒂股的"极度低估"标准。

## 五、支柱一：存量资产垫评估

### 5.1 T0 计算

```
现金池 = 现金等价物(10,549) + 短期理财(50) = 10,599 百万 CNY
T0 = 现金池 − 总负债(6,798) = 3,801 百万 CNY
T0_NAV = 3,801 / 总股本(2,926) = CNY 1.30 / 股
当前股价 = HK$10.40 ≈ CNY 9.47（按汇率0.91）
判定：T0_NAV(1.30) << 股价(9.47) → T0 不达标
```

### 5.2 T1 计算

```
有息负债 = 短期借款(1,203) + 长期借款(0) = 1,203 百万 CNY
T1 = 现金池(10,599) − 有息负债(1,203) = 9,396 百万 CNY
T1_NAV = 9,396 / 2,926 = CNY 3.21 / 股
判定：T1_NAV(3.21) << 股价(9.47) → T1 不达标
```

### 5.3 T2 计算

```
T2 = 现金等价物×1.0(10,599) + 应收账款×0.85(5,272×0.85=4,481)
   + 存货×0.7(82×0.7=57) + 其他流动资产×0.5(4,545×0.5=2,273)
   − 总负债(6,798)
   = 10,599 + 4,481 + 57 + 2,273 − 6,798 = 10,612 百万 CNY
T2_NAV = 10,612 / 2,926 = CNY 3.63 / 股
判定：T2_NAV(3.63) << 股价(9.47) → T2 不达标
```

存货折扣系数选取理由：IT服务公司存货极少（仅CNY 82百万），主要为硬件设备，取0.7。

### 5.4 特殊科目处理

- 受限现金：⚠️ 需查阅年报附注确认，在线数据源未单独披露
- 合同负债：IT服务行业合同负债不具备"准现金"属性，不加入现金池
- 租赁负债：⚠️ 可能包含在总负债中（IFRS 16），但不影响结论（三级均远不达标）
