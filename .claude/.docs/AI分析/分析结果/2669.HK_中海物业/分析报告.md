## 二、商业模式深度扫描

### 2.1 赚钱逻辑（Core Business Model）

| 维度 | 分析内容 |
|:-----|:---------|
| 核心主业 | 物业管理服务（住宅+商业+公建），覆盖中国内地、香港及澳门。管理面积持续增长，2024年在管面积约3.8亿平方米 |
| 收入构成 | 物业管理服务（基础物管费）为核心收入来源，占比约70%+；增值服务（社区增值、工程服务、城市运营等）占比约25-30%。城市运营业务为新增长点，录得收入及盈利双增长 |
| 盈利模式 | 按面积收取物管费（酬金制+包干制并存），赚取管理费差价；增值服务赚取服务溢价和工程利润 |
| 价值链位置 | 下游服务端，依托母公司中国海外发展（0688.HK）的地产开发获取大量在管面积，同时积极拓展第三方项目 |
| 客户结构 | To C（住宅业主）+ To B（商业物业）+ To G（政府公建/城市运营）。关联方收入占比约13.4%（FY2024关联方物管收入18.8亿/总收入140.2亿） |

### 2.2 周期性判别（Cyclicality Assessment）

| 维度 | 判断 |
|:-----|:-----|
| 行业周期属性 | **弱周期**。物业管理属于刚性需求服务，业主入住后持续缴纳物管费，收入具有高度经常性和可预测性 |
| 判定依据 | FY2022-2024收入持续增长（FY2022→FY2023→FY2024），即使在地产行业深度下行期间仍保持正增长。收入波动率极低 |
| 当前周期阶段 | 行业处于整合期。地产下行导致新盘交付放缓（合约转化速度下降），但存量物管面积提供稳定收入底盘。头部物管公司通过收并购和第三方拓展逆势扩张 |
| 对烟蒂股分析的影响 | 弱周期属性意味着资产估值稳定性高，不需要对存货/应收施加额外周期折扣。但当前地产下行对增值服务（尤其工程类）毛利率形成压力 |

### 2.3 提价权分析（Pricing Power）

| 维度 | 评估 |
|:-----|:-----|
| 定价能力 | **介于两者之间，偏价格接受者**。物管费受政府指导价和业委会约束，提价困难；但高端物业和增值服务有一定溢价空间 |

| 通胀环境下的表现 | 中性偏负面。人工成本随通胀上升，但物管费调整滞后且幅度有限 |

## 五、支柱一：存量资产垫评估

> 汇率假设：1 港元 ≈ 0.92 人民币（即 1 人民币 ≈ 1.087 港元）
> 市值：140.6亿港元 ≈ 129.4亿人民币
> 总股本：3,283,960,460股

### 5.1 T0 计算（基于FY2024年报数据）

```
现金池 = 现金及银行结余 5,803M + 受限制银行存款 9M = 5,812M 人民币
总负债 = 6,831M 人民币
T0 = 5,812M − 6,831M = −1,019M 人民币（负值）
T0_NAV = −1,019M / 3,284M股 = −0.31 人民币/股
当前股价 = 4.28港元 ≈ 3.94人民币
判定：T0_NAV < 0 → ❌ T0严重不达标（现金池不足以覆盖总负债）
```

### 5.2 T1 计算

```
有息负债 = 银行借款 50M + 租赁负债(53.6M + 68.6M) = 172M 人民币
T1 = 现金池 5,812M − 有息负债 172M = 5,640M 人民币
T1_NAV = 5,640M / 3,284M股 = 1.72 人民币/股 ≈ 1.87港元/股
判定：T1_NAV 1.87港元 < 股价 4.28港元 → ❌ T1不达标
T1总额 5,640M人民币 ≈ 61.3亿港元 < 市值 140.6亿港元
```

### 5.3 T2 计算

```
物管行业存货（停车位+工程物料）折扣系数取0.7
T2 = 现金等价物×1.0(5,803M) + 应收账款×0.85(2,595×0.85=2,206M)
     + 存货×0.7(653×0.7=457M) + 其他流动资产×0.5((1,157+152+848)×0.5=1,079M)
     − 总负债(6,831M)
   = 5,803 + 2,206 + 457 + 1,079 − 6,831
   = 2,714M 人民币
T2_NAV = 2,714M / 3,284M股 = 0.83 人民币/股 ≈ 0.90港元/股
判定：T2_NAV 0.90港元 < 股价 4.28港元 → ❌ T2严重不达标
T2总额 2,714M人民币 ≈ 29.5亿港元 < 市值 140.6亿港元
```
