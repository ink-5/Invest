## 核心思想

"烟蒂股"聚焦于企业当前时间截面的可变现资产，核心逻辑是——**"现在买下的资产，已经值得"**。

**三大支柱**：
1. **资产安全** — 现金资产显著高于市值（T0/T1/T2分级）
2. **经营安全** — 低资本开支与正自由现金流
3. **兑现安全** — 明确的派息或资产处置路径（A/B/C三种子类型）

## 财报与公告获取（港股强制执行）

**港股标的在分析启动后、数据提取之前，必须通过 `hkex-fetcher` skill 完成以下获取，作为分析的主数据源。** 严禁仅依赖第三方平台汇总数据。

具体 API 操作参见 `F:\Invest\.claude\skills\hkex-fetcher\SKILL.md`。

### 强制获取清单

| # | 类型 | 数量 | hkex-fetcher 参数 | 说明 |
|:-:|:-----|:----:|:------------------|:-----|
| 1 | 年报 | 最近 2 份 | `t1code=40000&t2code=40100` | 覆盖最近两个完整财年 |
| 2 | 中期报告 | 最近 2 份 | `t1code=40000&t2code=40200` | 覆盖最近两个半年度 |
| 3 | 最近1个月公告 | 全部 | `t1code=10000&t2code=-2&fromDate={30天前}&toDate={今天}` | 捕捉最新动态 |

### 执行步骤

```
1. 通过 hkex-fetcher prefix API 获取 stockId
2. 分别搜索年报（t2code=40100）和中报（t2code=40200），各取最新2条结果
3. 搜索最近1个月全部公告（t1code=10000，fromDate=当前日期-30天）
4. 下载PDF到标的文件夹，命名规则：
   - 年报：FY{财年}_年报.pdf（如 FY2024_年报.pdf）
   - 中报：FY{财年}_中期报告.pdf（如 FY2024_中期报告.pdf）
   - 公告：按原标题保存（如 盈利预警.pdf、须予披露的交易.pdf）
5. 同步更新 参考资料.md
6. 读取最新年报和中报PDF，提取关键财务数据
```

### 从财报PDF中必须提取的数据

年报/中报PDF是分析的主数据源，必须从中直接提取以下科目（而非依赖第三方平台）：

- 资产负债表全部关键科目（现金、应收、存货、流动资产、总资产、各类负债、净资产等）
- 受限现金/质押资产（附注）
- 商誉及减值测试（附注）
- 应收账款账龄分析（附注）
- 关联方交易明细（附注）
- 或有负债与资本承诺（附注）
- 对外担保（附注）
- 政府补贴金额（附注或利润表）
- 审计意见（审计师报告页）
- 经营性现金流、资本开支、自由现金流（现金流量表）
- 每股股息、派息率（董事会报告或利润分配方案）

### 最近1个月公告的用途

公告用于补充 Fact Check 和事件驱动分析的时效性信息：

- 盈利预警 / 盈喜 → 影响盈利预期和股息可持续性判断
- 大股东增减持 → Fact Check #18 管理层诚信 + C型事件信号
- 关联交易 / 重大资产处置 → Fact Check #14 + C1a事件催化
- 审计师变更 → Fact Check #17 一票否决项
- 回购记录 → C1b回购催化评估
- 私有化/要约收购 → C1c事件驱动
- 股息/分红公告 → A型股息可持续性验证

## 分析流程概要

详细执行步骤见 [烟蒂股分析Prompt.md](references/烟蒂股分析Prompt.md)，核心流程：

```
Step 0: 财报与公告获取（港股强制执行）
  → 通过 hkex-fetcher 获取最近4次财报（2年报+2中报）+ 最近1个月公告
  → 下载PDF到标的文件夹，更新参考资料.md
  → 读取最新年报和中报PDF，提取关键财务数据和附注信息
Step 1: 数据提取（从年报/中报PDF直接提取）+ 联网搜索实时数据（股价、市值、PB等）
Step 2: 商业模式深度扫描
Step 3: 管理层与治理分析
Step 4: 三大支柱评估（资产垫T0/T1/T2 + FCF + 兑现逻辑A/B/C）
Step 5: 子类型专项评估
Step 6: Fact Check 21项验证（结合年报附注原文数据 + 最近1个月公告）
Step 7: 操作建议与收益计算
Step 8: 生成完整报告
```
